Made with Bluefish HTML editor.

AKM started out as an exercise to help me learn Java because the software I support in my day job is migrating from C/C++/Tcl/Fortran/Python/Perl/<pick a language> to Java and I really needed to learn it. The target environment is Java/SWT/Eclipse RCP, which is why the GUI library currently used is SWT. JFace isn't used since I won't be using it at work and didn't want to take the time to learn it. I think a better choice for this project going forward is qt4 instead of SWT because it's faster, is more mature, and has some basic features that SWT lacks (such as filtering and sorting on tables, separation of the model from the view, etc.). AKM started out as a simple command line utility that would query a specified Quake 3 server (since I mostly play Q3) and display the results on stdout. I then added master server polling and queried all servers in the resultant list. SWT was added to create the display, but the underlying query structure stayed in place, which is why the main display updates in a single batch instead of displaying servers as results are returned. AKM's Quake 3 heritage is still obvious in much of the code. AKM is patterned after Yahoo's now-defunct All-Seeing Eye, though it has some features that ASE did not, plus it will run on Linux and OS X in additional to Windows.
